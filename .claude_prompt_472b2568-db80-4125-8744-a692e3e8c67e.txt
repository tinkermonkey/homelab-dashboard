**Title**:
Phase 6: Bot console rail

**Description**:
Build the collapsible bot console rail with bot tabs, message thread display (including tool-call blocks and suggestion chips), and the message composer with keyboard shortcuts. Visibility and active bot persist to localStorage.

## Requirements
- ChatRail (custom, wraps Heimdall Drawer): 380px right-docked, collapsible via topbar bot icon and internal close button; visibility persists
- Rail header: 'Bot console' label + cyan status dot, monospace subtitle, history/settings/close icon buttons
- Bot tab row: 4 tabs (lab-bot, ops-bot, watch-bot, sync-bot); each shows monospace name, 5px status dot, role in monospace caps; active tab has cyan bottom border; active bot persists
- Chat thread: day dividers in monospace with flanking 1px lines; messages with 22px avatar, sender name + role badge + timestamp, body text
- Tool-call blocks: lightning icon, tool name in monospace caps, status badge (e.g. 'completed in 1.4s' in emerald), KV output lines
- Suggestion chips: monospace pill buttons; clicking submits the chip text as a user message
- Composer: scope line ('talking to [bot-name]'), '+ context' and '/cmd' pills, textarea, keyboard hints; Enter sends (appends to thread + clears textarea), Shift+Enter inserts newline; cyan send button
- Topbar bot icon highlighted in cyan when rail is open

## Design Guidance
